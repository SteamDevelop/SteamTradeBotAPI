# SteamTradeBotAPI
steam bot api，auto trade csgo dota2 skins

我们对外提供接口，通过接口可以调用Steam皮肤交易服务，完成Steam皮肤自动化交易。

## 接口列表如下：
- Steam机器人状态查询
- 控制Steam机器人上下线
- 发起交易接口
- 查询交易状态
- 取消交易
- 报价状态回调
